Changedown for Word

Open this pane from Microsoft Word.

This page is the hosted task pane for the Word add-in. To connect it to your local Changedown MCP bridge, run the setup command from the project you want agents to work in.

npx @changedown/cli@latest word start
  1. Run the command above in your terminal.
  2. It installs/trusts the Office local developer certificate, downloads the hosted manifest, configures detected agents, and sideloads Word.
  3. Open the Changedown pane in Word; it will connect to https://127.0.0.1:39990 when your agent starts Changedown MCP.
View on GitHub Hosted manifest

Already running locally? Try npx @changedown/cli@latest word doctor to check manifest validation, MCP health, CORS, and private-network access.
