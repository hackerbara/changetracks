ChangeDown for Word

Open this pane from Microsoft Word.

This page is the hosted task pane for the Word add-in. To connect it to your local ChangeDown MCP bridge, run the setup command from the project you want agents to work in.

npx changedown word start
  1. Run the command above in your terminal.
  2. It downloads the hosted manifest, starts the local MCP bridge, and sideloads Word.
  3. Open the ChangeDown pane in Word; it will connect to https://127.0.0.1:39990.

Already running locally? Try npx changedown word doctor to check certificates, manifest validation, MCP health, CORS, and private-network access.
